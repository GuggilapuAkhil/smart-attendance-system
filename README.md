This project automates attendance management using **Face Recognition** technology.  
It is built with **Python**, **OpenCV**, and **Machine Learning** techniques to detect and recognize faces in real time, record attendance automatically, and store it in CSV files.

---

## üß© Features

- Register new students through webcam image capture  
- Train the system to recognize faces automatically  
- Record attendance in real-time using facial recognition  
- Store attendance data in subject-wise CSV files  
- Simple and user-friendly GUI  
- View saved attendance details within the app
